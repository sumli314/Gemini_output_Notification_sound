# Gemini_output_Notification_sound
What is this?
About This is a Gemini monitoring script. It tracks the "Stop" button to detect if the AI is generating content. When the button vanishes, a silenceCounter buffers the check to confirm the AI has truly finished, then triggers an audio alert to notify you. 
這是為 Gemini 設計的監控腳本。透過追蹤「停止生成」按鈕來判斷 AI 是否在工作。當按鈕消失，系統會以 silenceCounter 進行緩衝檢查，確認 AI 輸出後，自動發出提示音提醒使用者。
How to use?(Only on Google Chrome)
Step1: Open Extensions Page: Type chrome://extensions/ into your browser's address bar and press Enter.
Step2: Enable Developer Mode: Locate the Developer mode toggle switch in the top-right corner of the page and turn it on.
Step3: Load Unpacked Extension: Click the Load unpacked button that appears in the top-left corner.
Step4: Select Directory: Browse to and select the folder containing your extension's source code, then click Select Folder (or OK) to complete the installation.
Step1: 開啟擴充功能管理頁面： 在網址列輸入 chrome://extensions/ 並按下 Enter 鍵。
Step2: 啟用開發人員模式： 找到頁面右上角的「開發人員模式 (Developer mode)」切換開關，將其開啟。
Step3: 載入資料夾： 點擊左上角出現的「載入未封裝項目 (Load unpacked)」按鈕。
Step4: 選擇目錄： 瀏覽並選取包含您擴充功能原始碼檔案的資料夾，最後點擊「選取資料夾」或「確定」即可完成安裝。
